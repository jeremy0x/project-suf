A modern, responsive website for Shape Up Fitness, a premier fitness center in Akure, Nigeria. Built with React, TypeScript, and Tailwind CSS. Visit the live site at [shapeupfitnessonline.com](https://shapeupfitnessonline.com).

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion
- React Router

## Getting Started

1. Fork the repository

2. Clone your fork:

```bash
git clone https://github.com/your-username/project-suf.git
cd project-suf
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Build for production:

```bash
npm run build
```

## Project Structure

```
src/
├── components/     # Reusable UI components

├── pages/         # Page components
├── styles/        # Global styles
```
